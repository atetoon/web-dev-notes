# Conditional Statements

## Basic Forms

| Statement | Purpose |
| --- | --- |
| `if` | Runs if condition is true |
| `if...else` | 2 outcomes |
| `else if` | Multiple conditions |

## Comparison Operators

```
<  >  <=  >=  ===  !==
```

## Logical Operators

| Operator | Meaning |
| --- | --- |
| `&&` | Both true |
| `\|\|` | At least one true |
| `!` | Reverse (true ↔ false) |

## Ternary (Shortcut)

```js
condition ? value1 : value2;
```

## Switch

```js
switch(x) {
  case 1:
    break;
}
```
