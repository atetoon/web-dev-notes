# JavaScript Basics

## Console & Comments

```js
console.log("Hello");  // print output

// single-line comment
/* multi-line comment */
```

## Data Types (8)

```
Number, BigInt, String, Boolean, Null, Undefined, Symbol, Object
```

| Type | Example |
| --- | --- |
| Number | `20` (no quotes) |
| String | `'text'` |

## Operators

```js
+  -  *  /  %
```

`+` with strings → concatenation.

## Properties & Methods

```js
'hello'.length          // property
'hello'.toUpperCase()   // method
```

Use `.` (dot operator) to access both.

## Built-in Objects

```js
Math.random();   // random number
Math.floor();    // round down
```
